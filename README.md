## 适用场景

- 查询女足球员经历（成长故事、巅峰时刻、转型挑战）
- 了解顶级教练的领导力哲学
- 研究全球女足发展历史与文化差异
- 为写作、演讲、教学获取真实素材
- 了解中国女足的体制背景与国际对比

### 方式一：通过 ClawHub（推荐）
